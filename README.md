# Salad Fork Printer
Repo for configs for my printer. No electrons flowing through the printer so this is a draft as a config can be. Only validation I've done so far is bench testing, but I'm expecting my limits, mins and maxes, and other macros to need to be adjusted as needed. Sharing this anyway. 

## My Customizations
* BTT Manta E3EZ + CB2 for operations
* Phaetus rapido v2 HF
* Fysetc h36 tool head board
* Cartographer3d v4 for z probe
* Galileo 2 SA extruder
* Opto22 SSR
* Using anthead 
* Using a crossbow for cutting
* Using pure Armbian for CB2
* Planning on using gt1.5 pulleys and belts
* Heavy use of funssor parts
  * Belt A & B Mounts + rear adjustment
  * bed mount
  * Z mounts
  * lightweight X gantry
  * extrusion braces
